# Kamome for iOS
## Build SDK

1. Open Kamome.xcworkspace using Xcode
1. Run "Universal > Generic iOS Device"

## Run Sample App

1. Open Kamome.xcworkspace using Xcode
1. Build SDK before running the sample app
1. Run app
    
    **Swift** Run KamomeSwift > (Your Device)  
    **Objective-C** Run Kamome / (Your Device)
